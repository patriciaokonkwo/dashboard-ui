# Setup Instructions

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/patriciaokonkwo/dashboard-ui.git
   cd dashboard-ui
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## Build
To create a production build, run:
```bash
npm run build
```
